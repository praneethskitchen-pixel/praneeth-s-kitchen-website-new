# Public Assets Folder

Place your custom static files here.

### Adding Your Custom Logo
To use your custom logo in the application:
1. Name your logo image file `logo.png`.
2. Save it directly in this `/public` folder (so its path is `/public/logo.png`).
3. The website will automatically detect the file and render it in the header, footer, hero section, and receipts!

*Note: If no `/public/logo.png` is found, the application gracefully falls back to a hand-crafted, responsive golden SVG logo.*
